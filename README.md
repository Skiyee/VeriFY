# VeriFy

**VeriFy**(Verify for you) 是一个支持多类型即时串行校验器

> 该校验器是在async-validtor的基础上重构的，目的是为了适应自己的项目

## 特点

- **数据验证**：VeriFy 支持基础类型，并且拓展常见的类型

- **轻松集成**：VeriFy 支持多种引入方式

- **双端支持**：VeriFy 支持云端和本地端
